定期検診は親ができる「お子さんへの最高のプレゼント」

　

　この項に関して最後に付け加えておけば、親御さんは、子どもさんに対して歯磨きの習慣と同じように、定期検診を欠かさずに受ける習慣を付けてあげてほしいと思います。

　大人になり、むし歯だらけになってからでは遅すぎますし、また、むし歯がなくても歯周病予防のための検診を受けておいたほうがいいからです。

　むし歯がない状態の定期検診での受診になれば、痛い思いもしなくて済むのでトラウマになることもないと思いますが、むし歯になってから連れてこられれば必ず治療のために行くことになるため、少なからず歯医者さん＝痛いところというイメージがついてしまいます。
