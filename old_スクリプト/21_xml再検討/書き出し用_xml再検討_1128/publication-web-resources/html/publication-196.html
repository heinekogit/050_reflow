　また、当たり前と思っていることが当たり前でないのは、先進国であっても同じです。

　世界には、公的医療保険が充実していない国々がたくさんあります。そうした国では、民間の医療保険に入っていなければ高額な治療費を払えないため、よほど具合が悪くない限り病院へ行くのも躊躇することになります。
