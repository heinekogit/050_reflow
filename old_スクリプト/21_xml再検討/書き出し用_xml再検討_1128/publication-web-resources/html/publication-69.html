自分の希望が、歯医者さんにちゃんと伝わっているかを質問して確認する

　

　その反対に、自分がスタッフに伝えたことを歯医者さんが知らなかったらどうでしょう？　もしくは、歯医者さんが前回までの話の内容を忘れている様子なら？

　「一体何のためのカウンセリングだったのか？」となりますよね。

　特に、過去の病歴や体質などに関することならば、重大なリスクにもなりかねません。患者さんは、初診時にカウンセリングを受けただけ（話をしただけ）で満足せず、その後も歯医者さんと積極的にコミュニケーションを取りましょう。

　自分の状況や希望がきちんと歯医者さんに伝わっているか（憶えているか）を折に触れて確認しましょう。疑問点や不安なことはどんどん質問し、自分の希望を伝えてください。
